## Affix Props (参数)
| Name | Type | Default | Description |
| --- | --- | --- | --- |
| offset | `number` | `0` | 距离达到指定偏移量后触发 |
| target | `string` | `window` | 指定容器 |
| position | `'top'` \| `'position'` | `'top'` | 固定的相对方向 |
| z-index | `number` | `998` | z-index 值 |
